    Struct Output

    A response to user. Default content-type is "text/html".

    struct Output ; 
    // Set status code to 404
output.status = 404

// Send a response. Same as: output.write("Sorry, page not found.");
output ~= "Sorry, page not found.";

    Properties

    NameTypeDescription
    status[get] ushortRead status.
    status[set] ushortSet response status. 200 by default.

    Methods

    NameDescription
    addHeader (key, value) You can add a http header. But you can't if body is already sent.
    headersSent () Are headers already sent?
    opAssign (v) Mute/unmute output. You must set this before writing any data to output
    opOpAssign (data) Syntax sugar. Easier way to write output.
    sendHeaders () Force sending of headers. You can't add/edit headers after this.
    serveFile (path, guessMime) You can reply with a file. Automagical mime-type detection.
    setCookie (c) Add or edit a cookie. You can't if body is already sent. To delete a cookie, use cookie.invalidate() and then setCookie(cookie)
    setTimeout (max) Override timeout for this request
    write (data) Write data to output. You can write as many times as you want.
